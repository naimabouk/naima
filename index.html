compus Network

compus Network Campus description: in this topology we have a full campus network layered with the core, the Distribution and the access layer. As shown in the topology the access layer allows end terminal to access the network, while the Distribution layer is responsible of the connection between the access layer to the core layer and the core layer provides a high-speed connection between different distribution layer devices. We configured link aggregation, layer 2 and layer 3 redondancy.
